# Bible Text Quote Recognition Refactoring

## Current Problem
- Text quote recognition fails for certain formats
- Specific issues:
  1. Requires manual conversion of book names (e.g., "Salmos" to "Sal")
  2. Cannot handle multi-verse references like "Sal. 1:1, 4, 7"

## Desired Outcome
- Flexible bible text recognition
- Support for multiple language variations
- Handle complex verse references
